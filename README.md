# geojson-visualizer
**Earthquake Visualization (`earthquake.html`)**:
   - Visualizes earthquake data and Japan boundaries.
   - Displays a table with earthquake details (`id`, `magnitude`, `timestamp`).
   - Includes a button to sort the table by magnitude.
